# Pitalak Backend API 🍬

## Pitalac® (в разработке)
RESTful API для SPA-приложения по продаже полезных десертов и правильного питания.
Проект ориентирован на прозрачную работу с нутриентами (состав, БЖУ, рекомендации) и удобные пользовательские сценарии, включая OTP-авторизацию.

## Возможности проекта:
- Каталог продуктов с ингредиентами, нутриентами и изображениями
- Динамический расчёт БЖУ (белки, жиры, углеводы)
- Корзина покупок и управление заказами
- Адреса пользователей и данные доставки
- Авторизация по одноразовому коду (OTP)
- Stateless RESTful API с JWT-аутентификацией
- Административная панель для управления контентом и заказами
- Автоматически генерируемая документация API (Swagger / Redoc)
